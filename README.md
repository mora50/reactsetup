# React NextJS SETUP - TypeScript

A simple setup of react next app, that includes: styled-components, eslintconfig, prettierconfig and svg babel plugin for more optimization

## How to install

1. Install with npm

```sh
npm install
```

## How to run

1. Run the project

```sh
npm run dev
```
